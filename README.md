# AI-Chatbot
[C++] A Simple Chatbot

___

## How to use it

Compile the project with `make`

Execute the project with `./AI-Chatbot`

Now you can speak with the bot,

You have to speak a long time to learn it how to speak

To quit a conversation, say `quit()`
